# python目录结构

- log 日志目录

- conf 配置目录 .xml 文件

- src 核心代码

- lib 第三方库

- docs 文档库

- bin 启动入口,存放可执行文件 .sh

- tests 存放测试代码

- bin binary是二进制执行文件目录

- main.py

- setup.py 运行环境配置

- requirements.txt python版本等

- README.md 帮助文档
